On this portal, you can test out your C++ knowledge in different areas, or just practice. In this portal you will find usual C++ interview questions with code, where you have to guess what will be the output. The questions are separated into several groups, so you can only test or practice certain areas.

How to use:

  • Each code snippet has a question, which is likely to summarize the important fact, that the code snippet needs. So during practice, you may only need the question to remember an important C++ fact.
  • Below the question, you can see the code snippet.
  • You have to guess what will be the output.
  • If you want to see the result, click on the code, that will turn the card around, showing some comments on the code and showing the output.
  • Some codes will lead to compile error, or runtime error, these will be indicated in the output, or in the comments.
